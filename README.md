# teamrepair
**Repair teamviewer to debian 9 linux
¿Como empezar a realizar la reparacion?, solo abre la consola y ejecuta esto:
git clone https://github.com/juanma386/teamrepair/ && chmod -Rv 755 * && cd teamrepair && sh *.sh
